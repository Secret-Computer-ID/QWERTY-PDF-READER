# Frontend - QWERTY PDF Reader

React + TypeScript + Vite setup untuk aplikasi web PDF Reader.

## Setup

```bash
cd frontend
npm install
npm run dev
```

## Dependencies

- React 18+
- TypeScript
- Vite
- Tailwind CSS
- Framer Motion
- PDF.js
- Zustand
- React Query

## Struktur Folder

```
src/
├── components/    # React components
├── pages/         # Page components
├── hooks/         # Custom hooks
├── services/      # API services
├── store/         # State management
├── types/         # TypeScript types
└── utils/         # Utility functions
```

## Scripts

- `npm run dev` - Development server
- `npm run build` - Build production
- `npm run preview` - Preview build
- `npm run lint` - Lint code
